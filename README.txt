Мой проект называеться 'вкусно и вопросительный знак' 
Это приложение с графическим интерфейсом позволяющее заказывать еду через приложение. 
С помощью этого приложения пользователь с лёгкостью сможет добавить позицыы в приложение
Администратор в свою очередь может изменять позиции добавлять новые и удалять старые не влезая в сам код
